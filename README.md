# Computer Vision Experimentation

This repository is used to experiment with various CV ressourses.

Thanks to Adrian from [pyimagesearch](https://www.pyimagesearch.com) for his tutorials and blog posts. Have a look at his [Resource Guide](cv_dl_resource_guide.pdf).

### Covered subjects

1. [Change detection](change_detection) (with [tutorial](https://www.pyimagesearch.com/2017/06/19/image-difference-with-opencv-and-python/))
2. [Face detection](face_detection) (with [tutorial](https://www.pyimagesearch.com/2018/02/26/face-detection-with-opencv-and-deep-learning/) - day 1)
3. [OpenCV basics](opencv_basics) (with [tutorial](https://www.pyimagesearch.com/2018/07/19/opencv-tutorial-a-guide-to-learn-opencv/) - day 2)
   * Load and display images
   * Rezise, rotate, blur and draw on images
   * Convert images to grayscale
   * Perform edge detection
   * Threshold a grayscale image
   * Find, count, and draw contours
   * Conduct erosion and dilation
   * Mask of an image
